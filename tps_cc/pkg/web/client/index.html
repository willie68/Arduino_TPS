TPS_CC, the Arduino TPS Cross Compiler

Beschreibung

Mit diesem kleinen Tool können Sie eine TPS-Datei in ein Arduino IDE-Projekt konvertieren.
Mit der compile-Option können Sie die Datei für das gewünschte Board automatisch kompilieren. (über Board wählbar)
Als Ergebnis erhalten Sie eine ZIP-Datei mit der Arduino-Ino-Datei und den direkten Abhängigkeiten.
Wenn Sie das Projekt selbst compilieren möchten, besuchen Sie bitte http://www.rcarduino .de/ für das Arduino TPS-Projekt.
Das generierte Projekt hat keinen der verschiedenen Programmiermodi, es wird einfach nur die TPS-Datei ausgeführt.
Zum Generieren einer TPS-Datei können Sie meinen SPS-Emulator verwenden.
Wenn Sie Probleme mit diesem Tool haben, schreiben Sie mir einfach eine Nachricht oder verwenden Sie den Issue Tracker auf meiner github Arduino TPS-Projektseite

Description

With this little tool, you can convert an TPS File into a Arduino IDE project.
With compile option you can automatically compile the file for the desired Board. (Selectable via Board)
As result you will get a zip file with the Arduino ino File and the direct dependencies.
If you want to compile the project by yourself, please visit http://www.rcarduino.de/ for the Arduino TPS project.
The generated project will have none of the different programming modes, it will simply only run the TPS file.
For generating a TPS file you can use my SPS-Emulator.
If you have problems with this tool, simply drop me a notice or use the issue tracker on my github Arduino TPS Project page
